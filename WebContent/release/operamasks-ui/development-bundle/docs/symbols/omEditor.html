Class omEditor

作者: 李聪平.

HTML富文本编辑器.
特点：

  1. 在网页中使用的可见即所得HTML富文本编辑器
  2. 支持更换皮肤样式
  3. 支持文件上传
  4. 支持自定义工具条
  5. 支持自定义快捷键及其命令
工具条命令列表：
  1. 'Source'：源码
  2. 'Preview'：预览
  3. 'Print'：打印
  4. 'PasteText'：粘贴为无格式文本
  5. 'PasteFromWord'：从 MS Word 粘贴
  6. 'Undo'：撤销
  7. 'Redo'：重做
  8. 'Find'：查找
  9. 'Replace'：替换
  10. 'Bold'：加粗
  11. 'Italic'：倾斜
  12. 'Underline'：下划线
  13. 'Strike'：删除线
  14. 'Subscript'：下标
  15. 'Superscript'：上标
  16. 'RemoveFormat'：清除格式
  17. 'NumberedList'：编号列表
  18. 'BulletedList'：项目列表
  19. 'Outdent'：减少缩进量
  20. 'Indent'：增加缩进量
  21. 'Blockquote'：块引用
  22. 'JustifyLeft'：左对齐
  23. 'JustifyCenter'：居中
  24. 'JustifyRight'：右对齐

  26. 'BidiLtr'：文字方向为从左至右
  27. 'BidiRtl'：文字方向为从右至左
  28. 'Link'：插入/编辑超链接
  29. 'Unlink'：取消超链接
  30. 'Anchor'：插入/编辑锚点链接
  31. 'Image'：图象
  32. 'Flash'：Flash
  33. 'Table'：表格
  34. 'HorizontalRule'：插入水平线
  35. 'Smiley'：表情符
  36. 'SpecialChar'：插入特殊符号
  37. 'PageBreak'：插入分页符
  38. 'Iframe'：iFrame
  39. 'Styles'：样式
  40. 'Format'：格式
  41. 'Font'：字体
  42. 'FontSize'：大小
  43. 'TextColor'：文本颜色
  44. 'BGColor'：背景颜色
  45. 'Maximize'：全屏
  46. 'ShowBlocks'：显示区块
  47. 'About'：关于omEditor
示例：
<script type="text/javascript" >
$(document).ready(function() {
    $('#editor').omEditor({
        skin:'kama',
        toolbar:'Basic',
        filebrowserImageUploadUrl : '/operamasks-ui/omUpload.do?type=Images'
    });
});
</script>
<textarea id="editor" name="editor" class="editor"><p>Initial value </p></textarea>

属性

  • allowImageType

    Type:
      Array
    Default:
    ['jpg','gif','png']

    允许上传的图片类型。

    示例：

    $( "#editor" ).omEditor({ allowImageType:['jpg','gif','png','bmp'] });

  • basicEntities

    Type:
      Boolean
    Default:
    true

    是否在“源码”模式中对“所见即所得”模式里的HTML实体进行转义，包括:nbsp,gt,lt,amp;

    示例：

    $( "#editor" ).omEditor({ basicEntities: false });

  • blockedBrowserKeystrokes

    Type:
      Array
    Default:
    (查看该属性示例)

    定义一组快捷键列表用来阻止浏览器对这些按键的默认行为。

    示例：

    // 这是默认值
var config = {};
config.blockedBrowserKeystokes =
[
    OMEDITOR.CTRL + 66 /*B*/,
    OMEDITOR.CTRL + 73 /*I*/,
    OMEDITOR.CTRL + 85 /*U*/
];
$( "#editor" ).omEditor(config);

  • docType

    Type:
      String
    Default:
    '<!DOCTYPE HTML>'

    设置加载为HTML编辑器的内容时，要使用的DOCTYPE。

    示例：

    // Set the doctype to the HTML 4 (quirks) mode.
$( "#editor" ).omEditor({ docType: '<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">' });

  • enableTabKeyTools

    Type:
      Boolean
    Default:
    true

    允许上下文敏感的tab键行为，包括下列情况：

    当在表格里面按下tab键时:
    • 如果 TAB 被按下, 选择表格的下一个单元格，如果当前是表格的最后一个单元格则新建一行并focus到该行的第一个单元格
    • 如果 SHIFT+TAB 被按下, 选择表格的上一个单元格，如果当前是表格的第一个单元格则什么事都不做.

    示例：

    $( "#editor" ).omEditor({ enableTabKeyTools: false });

  • enterMode

    Type:
      Number
    Default:
    OMEDITOR.ENTER_P

    设置回车键的行为. 这也决定了在编辑器中的其他行为规则, 例如元素 <br>是否作为段落分隔时使用。 允许的值包括以下常量，以及它们的相对行为:

    • OMEDITOR.ENTER_P (1): 创建新的 <p> 段落;
    • OMEDITOR.ENTER_BR (2): 行被 <br> 元素分隔;
    • OMEDITOR.ENTER_DIV (3): 创建新的 <div> 块
    注意: 建议使用 OMEDITOR.ENTER_P 值，因为它的语义值和正确性。并且编辑器会优化这个值.

    示例：

    // 不建议使用的.
$( "#editor" ).omEditor({ enterMode: OMEDITOR.ENTER_BR });

  • filebrowserImageUploadUrl

    Type:
      String
    Default:
    ''(empty string = disabled)

    图象对话框中处理图片上传的服务端地址。默认为空，表示不支持图片上传。

    示例：

    $( "#editor" ).omEditor({ filebrowserImageUploadUrl:'/uploadImage.do?type=Images' });

  • height

    Type:
      Number
    Default:
    200

    编辑器的高度。注意: 不支持百分比

    示例：

    $( "#editor" ).omEditor({ height: 500 });

  • keystrokes

    Type:
      Array
    Default:
    (查看该属性示例)

    快捷键关联命令的列表。 列表中的每个元素是一个数组，其中第一项是按键，第二个是要执行的命令的名称。 注意：如果设置了这个属性值则默认值会被覆盖

    示例：

    // 下面是默认值.
var config={};
config.keystrokes =
[
    [ OMEDITOR.ALT + 121 /*F10*/, 'toolbarFocus' ],
    [ OMEDITOR.ALT + 122 /*F11*/, 'elementsPathFocus' ],

    [ OMEDITOR.SHIFT + 121 /*F10*/, 'contextMenu' ],

    [ OMEDITOR.CTRL + 90 /*Z*/, 'undo' ],
    [ OMEDITOR.CTRL + 89 /*Y*/, 'redo' ],
    [ OMEDITOR.CTRL + OMEDITOR.SHIFT + 90 /*Z*/, 'redo' ],

    [ OMEDITOR.CTRL + 76 /*L*/, 'link' ],

    [ OMEDITOR.CTRL + 66 /*B*/, 'bold' ],
    [ OMEDITOR.CTRL + 73 /*I*/, 'italic' ],
    [ OMEDITOR.CTRL + 85 /*U*/, 'underline' ],

    [ OMEDITOR.ALT + 109 /*-*/, 'toolbarCollapse' ]
];
$( "#editor" ).omEditor(config);

  • readOnly

    Type:
      Boolean
    Default:
    false

    编辑器是否只读。

    示例：

    $( "#editor" ).omEditor({ readOnly: true });

  • resizable

    Type:
      Boolean
    Default:
    true

    是否允许改变编辑器大小。

    示例：

    $( "#editor" ).omEditor({ resizable:true });

  • skin

    Type:
      String
    Default:
    'eac'

    编辑器的皮肤。可以是安装路径下的皮肤文件夹的名字，或者是用逗号分隔路径和文件夹的名字。

    示例：

    $( "#editor" ).omEditor({ skin: 'kama' });
$( "#editor" ).omEditor({ skin: 'myskin,/customstuff/myskin/' });

  • startupFocus

    Type:
      Boolean
    Default:
    false

    设置页面加载的时候编辑器是否默认focus。

    示例：

    $( "#editor" ).omEditor({ startupFocus:true });

  • startupMode

    Type:
      String
    Default:
    'wysiwyg'

    设置编辑器启动时加载的模式。 这取决于加载的插件。默认情况下，“所见即所得”(wysiwyg)和“源”(source)模式可供选择。

    示例：

    $( "#editor" ).omEditor({ startupMode:'source' });

  • toolbar

    Type:
      Array,String
    Default:
    'Full'

    工具箱（别名工具栏）的定义。它是一个工具栏的名称或数组。 editor默认定义了两个工具箱，名字分别为'Full'和'Basic'。

    示例：

    // 定义一个工具条只包含”源代码“、一个分隔符，”加粗“和”斜体“按钮 
var config = {};
config.toolbar =
[
    [ 'Source', '-', 'Bold', 'Italic' ]
];
$( "#editor" ).omEditor(config);
    // 加载名字为'Basic'的工具集.
$( "#editor" ).omEditor({ toolbar: 'Basic' });

  • uiColor

    Type:
      String
    Default:
    ''(empty)

    编辑器的界面颜色。并不适用于所有皮肤。

    示例：

    $( "#editor" ).omEditor({ uiColor:'#AADC6E' });

  • undoStackSize

    Type:
      Number
    Default:
    20

    要保持的撤销步骤的数量。此值设置越高则占用内存越大。

    示例：

    $( "#editor" ).omEditor({ undoStackSize:50 });

  • width

    Type:
      String,Number
    Default:
    '' (empty)

    编辑器的宽度，可设置具体的像素值或百分比。

    示例：

    $( "#editor" ).omEditor({ width: 850 });
    $( "#editor" ).omEditor({ width: '75%' });

方法

  • getData

    Signature:
      getData()

    获取editor的HTML内容。

    返回:
    omEditor的HTML内容

    示例：

    $('#editor').omEditor('getData');

  • getPlainText

    Signature:
      getPlainText()

    获取editor的内容。包含纯文本，不包含IMG、EMBED元素以及文本头尾的换行符和空格符。

    返回:
    omEditor的文本内容

    示例：

    $('#editor').omEditor('getPlainText');

  • insertHtml

    Signature:
      insertHtml(params)

    插入HTML内容。

    参数:
    params - 要插入的HTML内容

    示例：

    $('#editor').omEditor('insertHtml', '<a href="#">link</a>');

  • insertText

    Signature:
      insertText(params)

    插入文本内容。

    参数:
    params - 要插入的内容

    示例：

    $('#editor').omEditor('insertText', 'new text');

  • setData

    Signature:
      setData(params)

    设置editor的内容。

    参数:
    params - 要设置的内容

    示例：

    $('#editor').omEditor('setData','<a href="#">link</a>');

  • setReadOnly

    Signature:
      setReadOnly(params)

    设置editor为只读模式。

    参数:
    params - Boolean是否只读

    示例：

  • updateElement

    Signature:
      updateElement()

    更新<textarea>元素，取而代之的是在编辑器中当前的数据。

    示例：

    $('#editor').omEditor('updateElement');
