Class omButton
  • 预览
  • 属性
  • 方法
  • 事件
作者: 罗业刚.

按钮组件。类似于html中的button、input[type=submit]、input[type=button]，使用背景图片实现圆角，支持icon，可以只显示icon。

特点：

  1. 实现圆角
  2. 支持左icon和右icon，可同时出现左右icon，也可以只显示icon不显示label
  3. 按钮文字数目不限，也可以任意设置按钮宽度
  4. 支持input[type=button]、input[type=submit]、input[type=reset]、button、a五种标签形式
示例：
<script type="text/javascript" >
$(document).ready(function() {
    $('#bnt').omButton({
        icons : {left:'images/help.png',right:'images/edit_add.png'},
        width : 150,
        disabled : 'disabled',
        onClick : function(event){
            // do something
        }
    });
});
</script>

omButton(p)
参数:
p - 标准config对象：{}

属性

  • disabled

    Type:
      String
    Default:
    null

    是否禁用组件。可通过('#id').attr('disabled')判断按钮是否禁用。

    示例：

    $("#button").omButton({disabled: true});

  • icons

    Type:
      Object
    Default:
    null

    显示按钮图标，left表示左图标，right表示右图标，取值均为图片路径。

    示例：

    $("#button").omButton({
    icons: {
        left: 'images/help.png',
        right: 'images/edit_add.png'
    }
});

  • label

    Type:
      String
    Default:
    ""

    显示文本。label值可以写到dom元素里面，也可以设置在属性里面，设置到属性里面的优先级最高。

    示例：

    $("#button").omButton({label: "apusic"});

  • width

    Type:
      Number
    Default:
    null

    按钮宽度，设置固定宽度之后按钮将不会随文字的多少而改变。

    示例：

    width : 150

方法

  • changeIcons

    Signature:
      changeIcons(icons)

    设置按钮的icon属性。

    参数:
    icons - 图标路径

    示例：

    $('#btn').omButton('changeIcons',{
    left: 'images/help.png',
    right: 'images/edit_add.png'
});

  • changeLabel

    Signature:
      changeLabel(label)

    设置按钮的label属性。

    参数:
    label - 按钮文本

    示例：

    $('#btn').omButton('changeLabel','按钮label');

  • click

    Signature:
      click()

    触发点击事件。

    示例：

    $('#btn').omButton('click');

  • disable

    Signature:
      disable()

    禁用组件。

    示例：

    $('#btn').omButton('disable');

  • enable

    Signature:
      enable()

    启用组件。

    示例：

    $('#btn').omButton('enable');

事件

  • onClick

    function()

    单击按钮触发事件。

    示例

    onClick : function(){
   //do something
}
