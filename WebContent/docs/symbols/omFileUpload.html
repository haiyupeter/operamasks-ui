Class omFileUpload
  • 预览
  • 属性
  • 方法
  • 事件

文件上传.
特点：

  1. 支持限制上传文件的大小和类型
  2. 支持批量上传文件
  3. 内置进度条展示文件上传进度
  4. 支持选中，上传成功，上传失败等多种回调事件
  5. 可自定义上传按钮的背景图片和文字
示例：
//注意：这里的swf属性指定了处理上传的swf文件位置，必须设置。这个位置是相对于HTML页面路径的。
<script type="text/javascript" >
$(document).ready(function() {
    $('#file_upload').omFileUpload({
        action : '/operamasks-ui/omFileUpload.do',
        onComplete : function(ID,fileObj,response,data,event){
            alert('文件'+fileObj.name+'上传完毕');
        }
    });
});
</script>

//注意：input的id属性必须设置
<input id="file_upload" name="file_upload" type="file" />

方法

  • cancel

    Signature:
      cancel(index)

    取消上传文件。如果不设置index参数则取消队列里面的所有文件。

    参数:
    index - 文件在上传队列中的索引，从0开始

    示例：

    $('#file_upload').omFileUpload('cancel'); // 取消上传队列中的所有文件
$('#file_upload').omFileUpload('cancel',1); // 取消上传队列中的第2个文件

  • upload

    Signature:
      upload(index)

    上传文件。如果不设置index参数则上传队列里面的所有文件。

    参数:
    index - 文件在上传队列中的索引，从0开始

    示例：

    $('#file_upload').omFileUpload('upload'); // 上传队列中的所有文件
$('#file_upload').omFileUpload('upload',1); // 上传队列中的第2个文件

事件

  • onAllComplete

    function(data, event)

    所有文件上传完后触发。

    参数:
    data - 封装了文件上传信息的Object对象，包含两个属性：fileCount(文件上传队列中剩余文件的数量)，speed(文件上传的平均速度 KB/s)
    event - jQuery.Event对象。

    示例

    $('#file_upload').omFileUpload({onAllComplete:function(data,event){alert('所有文件上传完毕');}});

    data - 封装了文件上传信息的Object对象，包含两个属性：fileCount(文件上传队列中剩余文件的数量)，speed(文件上传的平均速度 KB/s)

    每个文件完成上传后触发。

  • onError

    function(ID, fileObj, errorObj, event)

    文件上传出错后触发。

    参数:
    ID - 文件ID
    fileObj - 封装了文件的信息的Object对象，包含五个属性：name(文件名)，size(文件大小)，creationDate(文件创建时间)，modificationDate(文件最后修改时间)，type(文件类型)
    errorObj - 封装了返回的出错信息的Object对象，包含两个属性：type('HTTP'或'IO'或'Security')，info(返回的错误信息描述)

    示例
