 17 /**
 18      * @name omGrid
 19      * @class 表格组件。类似于html中的table，支持后台数据源、分页、自动列宽、单选/多选、行样式、自定义列渲染等功能。<br/><br/>
 20      * <b>特点：</b><br/>
 21      * <ol>
 22      *      <li>使用远程数据源</li>
 23      *      <li>支持分页展现</li>
 24      *      <li>自动添加行号</li>
 25      *      <li>允许某列的宽度自动扩充（该列宽度等于表格总宽度送去其它列宽度之和）</li>
 26      *      <li>允许所有列自动缩放（自动缩放各列的宽度，使得它适应表格的总宽度）</li>
 27      *      <li>可以定制隔行样式，也可以根据记录的不同使用不同的行样式</li>
 28      *      <li>可以定制各列的显示效果</li>
 29      *      <li>可以设置表头和表体自动换行</li>
 30      *      <li>可以改变列宽</li>
 31      *      <li>提供丰富的事件</li>
 32      * </ol>
 33      * 
 34      * <b>示例：</b><br/>
 35      * <pre>
 36      * <script type="text/javascript" >
 37      *  $(document).ready(function() {
 38      *      $('#mytable').omGrid({
 39      *          height : 250,
 40      *          width : 600,
 41      *          limit : 8, //分页显示，每页显示8条
 42      *          singleSelect : false, //出现checkbox列，可以选择同时多行记录
 43      *          colModel : [    {header:'编号',name:'id', width:100, align : 'center'},
 44      *                          {header:'地区',name:'city', width:250, align : 'left',wrap:true},
 45      *                          {header:'地址',name:'address', width:'autoExpand',renderer:function(value,rowData,rowIndex){ return '<b>'+value+'</b>'; }}
 46      *          ],
 47      *          dataSource : 'griddata.json' //后台取数的URL
 48      *      });
 49      *  });
 50      * </script>
 51      * 
 52      * <table id="mytable"/>
 53      * </pre>
 54      * 
 55      * colModel配置项是列模型，它定义了表格与具体数据无关的展现。它是一个JSON数组，其中数组中每一个item（是一个JSON对象）代表表格中的一列。可以使用以下属性：<br/>
 56      * <ol>
 57      *      <li>header:String类型，表示表格的表头文字。无默认值。</li>
 58      *      <li>width:Number类型或'autoExpand'，如果是数字200表示该列宽度是200px。如果是'autoExpand'表示该列的宽度是表格总宽度减去其它各列宽度之和。默认值是60。</li>
 59      *      <li>align:'left'或'right'或'center'，表示表格的表头和表体文字的对齐方式。默认值是'left'。</li>
 60      *      <li>name:String类型，说明表体这一列要显示后台返回的数据记录（JSON对象）中的哪个属性的值，如name:'sex'表示这一列显示JSON对象的sex属性的值。无默认值。</li>
 61      *      <li>wrap:Boolean类型，说明这一列的表头和表体当内容太多时是否允许自动换行显示，如wrap:true表示这一列的表头和表体在显示时自动换行（比如该列宽度是100px但是表体某td内容宽度是120px时则该td的内容自动换行显示）。默认值是false。</li>
 62      *      <li>renderer:Function类型，允许在显示该列的值是进行客户端处理。无默认值（不做转换直接输出，比如id是'name'的列将会直接把JSON行数据中的name属性的值显示在表格的td中）。有时可能需要进行一些处理，比如：<br/>
 63      *          <ul>
 64      *              <li>该td中的字体要加粗可以使用
 65      *                  <pre>
 66      *      renderer:function(value , rowData , rowIndex){
 67      *          return '<b>'+value+'</b>';
 68      *      }
 69      *                  </pre>
 70      *              </li>
 71      *              <li>比如后台返回的数据中sex是0或1，页面中要转换为中文'男'和'女'可以使用
 72      *                  <pre>
 73      *      renderer:function(value , rowData , rowIndex){
 74      *          return value==0?'男':'女';
 75      *      }
 76      *                  </pre>
 77      *              </li>
 78      *              <li>比如一个后台返回的数据中score是分数，现要求100分输出为绿色的'满分'小于60分时输出红色的'不及格'其它的不变，可以使用
 79      *                  <pre>
 80      *      renderer:function(value , rowData , rowIndex){ 
 81      *          if(value==100){
 82      *              return value==100?'<font color="green">满分</font>';
 83      *          }else if(value<60){
 84      *              return '<font color="red">不及格</font>';
 85      *          }else{
 86      *              return value;
 87      *          }
 88      *      }
 89      *                  </pre>
 90      *              </li>
 91      *          <ul>
 92      *      </li>
 93      * </ol>
 94      *

 98      *     [
 99      *         {"address":"CZ88.NET ","city":"IANA保留地址","id":"1"},

102      *         {"address":"CZ88.NET ","city":"澳大利亚","id":"4"},
103      *         {"address":"CZ88.NET ","city":"泰国","id":"5"},
104      *         {"address":"CZ88.NET ","city":"日本","id":"6"},
105      *         {"address":"电信","city":"广东省","id":"7"},
106      *         {"address":"CZ88.NET ","city":"日本","id":"8"}
107      *     ]
108      * }
109      * </pre>

113      * <ol>

115      *     <li>多选（一次可以选择多行）表格：①如果该行还未被选中，先触发该行的onRowSelect事件监听，如果该行已经选中，则先触发该行的onRowDeselect事件监听②触发该行的onRowClick事件监听。</li>
116      * </ol>
117      *

700             if (this.loading) {
701                 return true;

717                         newPage = nowPage - 1;
718                     }
719                     break;
720                 case 'next':
721                     if (nowPage < totalPages) {
722                         newPage = nowPage + 1;
723                     }

731                         nv = nowPage;
732                     }
733                     if (nv < 1) {
734                         nv = 1;
735                     } else if (nv > totalPages) {
736                         nv = totalPages;
737                     }
738                     $('.pControl input', this.pDiv).val(nv);
739                     newPage = nv;
740                     break;
741                 default:
742                     if (/\d/.test(ctype)) {
743                         var nv = parseInt(ctype);
744                         if (isNaN(nv)) {
745                             nv = 1;
746                         }
747                         if (nv < 1) {
748                             nv = 1;
749                         } else if (nv > totalPages) {
750                             nv = totalPages;
751                         }
752                         $('.pControl input', el.closest('.om-grid')).val(nv);
753                         newPage = nv;
754                     }
755             }
756             if (newPage == nowPage) {
757                 return false;
758             }
759             //触发事件
760             if(this._trigger("onPageChange",null,ctype,newPage)===false){

762             }
763             //修改pageData
764             pageData.nowPage=newPage;
765             //翻页时去掉全选状态
