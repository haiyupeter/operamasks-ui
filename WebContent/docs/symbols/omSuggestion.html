Class omSuggestion
  • 预览
  • 属性
  • 方法
  • 事件

    Ajax提示组件。类似于google首页的搜索功能（在输入的同时下拉框里给出可用的提示，用户可以从里面选择一个）。
    将该功能添加到一个input输入框上，将允许用户在输入的同时可以快速地查找和选择所要的内容。当输入框得到焦点并输入字符时，该组件会将用户输入的内容以Ajax方式发送到服务器进行处理，服务器处理完后返回一个数据集，客户端将数据集显示成一个可选列表，用户可以从可选列表中很方便地选择自己所要查找的东西。
    目前该组件主要用于从远程URL取得数据（如果是本地数据的话，可以使用omCombo组件，它也有边输入边过滤的功能）。一般用于从大量数据中进行查找的场合，如百度搜索、google搜索、taobao商品搜索、邮件系统快速输入收件人。
    该组件有客户端缓存的功能，如输入a开始Ajax查找；再输入b（输入框内容是ab）再次Ajax查找；再删除b（输入框内容是a）将不进行Ajax查找，因为缓存中已经有key=a的缓存内容，将直接根据缓存内容来重构可选列表而不发送Ajax请求从服务器取数。如果不需要缓存可以将cacheSize参数设成0。如果要清除缓存可以调用clearCache()方法。
    实际应用中一般都要控制可选列表框中记录的数目（如百度搜索、google搜索、taobao商品搜索的可选列表中记录数都为10，有道搜索的可选列表中记录数为8），这个要由服务器进行控制，服务器返回数据时请不要返回得太多（比如从数据库中查询时一般使用TOP-N查询）。

特点：

  1. 可以使用普通数组，也可使用JSON数组
  2. 支持鼠标操作和键盘操作
  3. 支持数据的客户端缓存
  4. 提供丰富的事件
  5. 用户可定制数据的显示效果
  6. 用户可定制请求的发送与处理
  7. 支持跨域请求数据

示例：
<script type="text/javascript" >
$(document).ready(function() {
    $('#input1').omSuggestion({
        dataSource:'/suggestion.json',
        minChars :3,
        listMaxHeight:40
    });
});
</script>

<input id="input1"/>

omSuggestion(p)
参数:
p - 标准config对象：

属性

  • cacheSize

    Type:
      Number
    Default:
    10

    本地缓存的数目。组件针对每一次输入值进行缓存，如果缓存中存在输入域中的输入值，不再发送ajax请求取数。
    注意：该属性值必须为非负整数。设置cacheSize:0禁用缓存

  • clientFormatter

    Type:
      Function
    Default:
    无

    下拉框中每行显示内容的转换器。对dataSource进行格式化（注意：如果dataSource返回的是非普通数组(具体请看dataSource属性的描述)一定要写clientFormatter属性进行格式化）。

    示例：

    //对于非普通record一定要写这个属性
clientFormatter:function(data,index){
        return '<b>'+data.text+'</b>(共找到'+data.count+'条记录)';
}

//对于普通的record也可以写这个属性
clientFormatter:function(data,index){
        return '<span style="color:red">'+data+'</span>;
}

  • crossDomain

    Type:
      Boolean
    Default:
    false

    Ajax请求是否需要跨域（从本页面所在的网站以外的地方取数）。注意：跨域请求时后台处理逻辑要进行特殊处理，具体请参考jQuery的JSONP相关知识。

  • dataSource

    Type:
      URL
    Default:
    无

    Ajax请求的URL路径，所有的请求将由此URL来处理，处理结果必须返回一个JSON数组。
    后台可以返回两种格式的数据：

    • 普通数组（如['a','b','c']）：可以不设置clientFormatter属性，也可以设置这个属性。
    • 非普通数组（如{"valueField":"text","data":[{"name":'张三',"sex":"男"},{"name":'李四',"sex":"女"},{"name":'王五',"sex":"男"}]}）：其中valueField表示回填时把data中每个JSON对象的哪个字段回填到输入框里。非普通数组时必须设置clientFormatter属性来告诉组件如果把这个JSON对象显示到下拉框里。

    示例：

    dataSource:'/operamasks-ui/getData.json'

  • delay

    Type:
      Number
    Default:
    500

    发送请求的延迟时间（单位是毫秒）。比如设成300，假设输入时每隔100ms输入一个字符，则快速输入1234时，只会在4输入完成后300ms才进行一次提示。注意：如果此属性值设成0或负数则不会延迟

  • disabled

    Type:
      Boolean
    Default:
    false

    是否禁用组件。如果禁用，则不可以输入，form提交时也将忽略这个输入框。

  • listMaxHeight

    Type:
      Number
    Default:
    300

    下拉框的最大高度（单位是px）。注意：由于浏览器的限制，这个属性的最小值是31，如果小于这个值时将看不到垂直滚动条

  • listWidth

    Type:
      Number
    Default:
    无

    下拉框的宽度。必须为数字。不设置时默认与输入框一样宽

  • method

    Type:
      String
    Default:
    'GET'

    Ajax请求时的方式，取值GET'或'POST'。

  • minChars

    Type:
      Number
    Default:
    1

    输入框输入字符数大于等于minChars时，才发送请求。注意：如果要页面一显示完就开始提示，可以设成0。

  • preProcess

    Type:
      Function
    Default:
    无

    数据成功响应后触发事件。
    一个Ajax请求成功（不出错误也不超时）后会先执行onSuccess事件的监听器，如果它返回false则不显示下拉框。如果没有监听器或者监听没有返回false则执行此preProcess预处理，处理结束后开始刷新并显示下拉框。

    示例：

    preProcess:function(text,data){
     $(data).each(function(){
        this.sex = this.sex==0?'男':'女';
    });
}

  • queryName

    Type:
      String
    Default:
    'key'

    发送Ajax请求时代表输入值的参数名。比如url是'fetchData.jsp?type=book'，queryName是'q'，当前输入的值是'abc'，则最终发送请求的url是'fetchData.jsp?type=book&q=abc'。

  • readOnly

    Type:
      Boolean
    Default:
    false

    组件是否只读。如果是只读，则不可以输入，form提交时将会包含这个输入框。

方法

  • clearCache

    Signature:
      clearCache()

    清空与此组件相关的缓存数据。每次提示后都会将结果集缓存（缓存的数目为config中配置的cacheSize），下次再需要对相同内容进行提示时会直接从缓存读取而不发送请求到服务器，如果需要忽略缓存而从服务器重新提示则可以调用此方法清除缓存。

    返回:
    无

    示例：

    $('#txt').omSuggestion('clearCache');

  • disable

    Signature:
      disable()

    禁用组件。

    示例：

    $('#myinput').omSuggestion('disable');

  • enable

    Signature:
      enable()

    启用组件。

    示例：

    $('#myinput').omSuggestion('enable');

  • getData

    Signature:
      getData()

    获取当前下拉框中的数据（服务器端返回的数据）。

    返回:
    Array[Json]

    示例：

    $('#txt').omSuggestion("getData");

  • getDropList

    Signature:
      getDropList()

    获取当前组件的下拉框。

    返回:
    jQuery Element

    示例：

    $('#txt').omSuggestion("getDropList").addClass('myselfClass');

  • setData

    Signature:
      setData(dataSource)

    设置新的请求地址，新地址表示参数的改变或者地址的改变。

    参数:
    dataSource -

    示例：

    $('#country').change(function() {
  var v = $('#country').val();
  $('#txt').omSuggestion("setData","../../../advancedSuggestion.json?contry="+v+"&province=hunan");
});

  • showMessage

    Signature:
      showMessage(message)

    在下拉框中显示一个提示信息（仅用于阅读，不可以通过快捷键或鼠标选择它）。

    参数:
    message - 要显示在下拉框中的消息

    示例：

    $('#txt').omSuggestion('showMessage','请求数据出错');

事件

  • onBeforeSuggest

    function(text, event)

    发送Ajax请求之前触发事件。注意：return false将会阻止请求发送。无返回值或return true将继续发送请求

    参数:
    text - 输入框里当前文本
    event - jQuery.Event对象

    示例

    $('#inputID').omSuggestion({
        onBeforeSuggest:function(text,event){
                if(text=='不文明用语'){
                        return false;//如果是不文明用语不进行提示
                }else{
                        return true;
                } 
        }
});

  • onError

    function(xmlHttpRequest, textStatus, errorThrown, event)

    Ajax请求出错时触发事件。

    参数:
    xmlHttpRequest - XMLHttpRequest对象
    textStatus - 错误类型
    errorThrown - 捕获的异常对象
    event - jQuery.Event对象

    示例

    $('#inputID').omSuggestion({
        onError:function(xmlHttpRequest, textStatus, errorThrown, event){
                $('#txt').omSuggestion('showMessage','请求出错。原因：'+errorThrown.message); 
        }
});

  • onSelect

    function(text, rowData, index, event)

    选择下拉框中一个后触发事件。

    参数:
    text - 输入框里当前文本
    rowData - 行记录，是Ajax请求返回的数据中的一行
    index - 当前行在下拉框所有行中的索引（第一行是0，第二行是1...）
    event - jQuery.Event对象

    示例

    $('#inputID').omSuggestion({
        onSelect:function(rowData,text,index,event){
                $('#searchbut').click(); //选择完后自动点击“查询”按钮
        }
});

  • onSuccess

    function(data, textStatus, event)

    Ajax响应回来时触发事件。

    参数:
    data - Ajax请求返回的数据
    textStatus - 响应的状态
    event - jQuery.Event对象

    示例

    $('#inputID').omSuggestion({
        onSuccess:function(data, textStatus, event){
                if(data.length==0){
                        $('#txt').omSuggestion('showMessage','无提示数据！');
                } 
        }
});

  • onSuggesting

    function(text, event)

    Ajax请求发送后响应回来前触发事件。

    参数:
    text - 输入框里当前文本
    event - jQuery.Event对象

    示例

    $('#inputID').omSuggestion({
        onSuggesting:function(text,event){
                $('#inputID').omSuggestion('showMessage','正在加载...'); 
        }
});
