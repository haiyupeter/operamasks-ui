Class omDialog
  • 预览
  • 属性
  • 方法
  • 事件

对话框组件。可以放置html。

特点：

  1. 方便地自定义按钮
示例：
<script type="text/javascript" >
$(document).ready(function() {
    $('#dialog').omDialog({
    });
});
</script>

<div id="dialog"/>

omDialog(p)
参数:
p - 标准config对象：{}

属性

  • autoOpen

    Type:
      Boolean
    Default:
    true

    对话框创建完成后是否自动打开。

    示例：

      $("#select").omDialog({autoOpen : true});

  • buttons

    Type:
      Array
    Default:
    []

    对话框中的按钮。此配置项为JSON数组。每个JSON对象具有 text 属性(配置按钮文字) 和 click 属性配置按钮触发时的回调方法。

    示例：

      $("#select").omDialog({buttons : [{
     text : "确定", 
     click : function () {...}
 }, {
     text : "取消", 
     click : function () {...}
 }]);

  • closeOnEscape

    Type:
      Boolean
    Default:
    true

    按下 Esc 键时是否关闭对话框。

    示例：

      $("#select").omDialog({closeOnEscape : true});

  • dialogClass

    Type:
      String
    Default:
    无

    对话框的样式。

    示例：

      $("#select").omDialog({dialogClass : 'class1'});

  • draggable

    Type:
      Boolean
    Default:
    true

    组件是否可拖动。

    示例：

      $("#select").omDialog({draggable : true});

  • height

    Type:

    Default:

方法

  • close

    Signature:
      close()

    关闭对话框.

    返回:
    支持链式操作，返回JQuery对象

    示例：

    var store = $("#select").omDialog('close');

  • isOpen

    Signature:
      isOpen()

    判断对话框是否已打开。

    返回:
    如果对话框已打开返回true，否则返回false

    示例：

    var isOpen = $("#select").omDialog('isOpen');

  • open

    Signature:
      open()

    打开对话框。

    返回:
    支持链式操作，返回JQuery对象

    示例：

    var store = $("#select").omDialog('open');
