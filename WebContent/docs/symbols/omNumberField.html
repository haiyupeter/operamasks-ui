Class omNumberField
  • 预览
  • 属性
  • 方法
  • 事件

数字输入框组件，只能输入数字，字符自动过滤掉。

omNumberField(p)

示例：

$('numberFielddiv').omNumberField({decimalPrecision:3});
参数:
p - 标准config对象：{}

属性

  • allowDecimals

    Type:
      Boolean
    Default:
    true

    是否允许输入小数。

    示例：

    $('#input').omNumberField({allowDecimals:true});

  • allowNegative

    Type:
      Boolean
    Default:
    true

    是否允许输入负数。

    示例：

    $('#input').omNumberField({allowNegative:true});

  • decimalPrecision

    Type:
      Number
    Default:
    2

    精确到小数点后几位。

    示例：

    $('#input').omNumberField({decimalPrecision:2});

  • disabled

    Type:
      Boolean
    Default:
    false

    是否禁用组件。

    示例：

    $('#input').omNumberField({disabled:true});

  • readOnly

    Type:
      Boolean
    Default:
    false

    是否只读。

    示例：

    $('#input').omNumberField({readOnly:true});

方法

  • disable

    Signature:
      disable()

    禁用组件。

    示例：

    $('#input').omNumberField("disable")

  • enable

    Signature:
      enable()

    启用组件。

    示例：

    $('#input').omNumberField("enable")

事件

  • onBlur

    function(value, event)

    在输入框失去焦点时触发的方法。

    参数:
    value - 当前输入框的值
    event - jQuery.Event对象

    示例

    $('#input').omNumberField({onBlur:function(value,event){alert('now the value is'+value);}});
