Class omTooltip
  • 预览
  • 属性
  • 方法

提示组件，当某个链接、表单、输入框等需要做功能向导提示的时候可以使用本组件。 特点：

  1. 提示内容多样化，可以是文字，html，页面dom节点还可以是异步加载的内容
  2. 异步加载内容的时候支持懒加载
  3. 最大限度节约资源，只有需要使用的时候才在页面加入dom元素
  4. 支持所有标准html元素
示例：
<script type="text/javascript" >
$(document).ready(function() {
    $('#tip').omTooltip({
        trackMouse : true,
        html : '<div style="color:red;">欢迎使用omTooltip组件</div>'
    });
});
</script>

属性

  • anchor

    Type:
      Boolean
    Default:
    false

    是否显示anchor，默认不显示，如果trackMouse为true，则显示在左上角，如果不跟随， 则根据region属性确定，箭头的方向永远指向被提示的目标。

    示例：

    $("#tip").omTooltip({anchor: false});

  • anchorOffset

    Type:
      Number
    Default:
    0

    anchor偏移度(px), 当设置anchor为top、button的时候，偏移度指相对于最左边的距离； 为left、right的时候偏移度指相对于最上的距离

    示例：

    $("#tip").omTooltip({anchorOffset:10});

  • contentEL

    Type:
      selector
    Default:
    null

    jquery选择器，可以获取页面的dom元素作为提示内容，格式为#ID、.className、tagName等，优先级比html属性低。

    示例：

    $("#tip").omTooltip({contentEL:'#aa'});

  • delay

    Type:
      Number
    Default:
    300

    鼠标移入目标区域之后弹出提示框的延迟时间(ms) ，同时也是隐藏延迟的事件。

    示例：

    $("#tip").omTooltip({delay: 300});

  • height

    Type:
      Number
    Default:
    'auto'

    提示框的高度。

    示例：

    $("#tip").omTooltip({height: 100});

  • html

    Type:
      String
    Default:
    null

    提示组件显示的内容，可以是html和普通字符,优先级比contentEL属性高。

    示例：

    $("#tip").omTooltip({html:'<span style="color:red;">我是omTooltip</span>'});

  • lazyLoad

    Type:
      Boolean
    Default:
    false

    ajax方式显示内容的时候是否延迟加载页面，必须配置了url才有效

    示例：

    $("#tip").omTooltip({url:'a.html',lazyLoad: true});

  • maxHeight

    Type:
      Number
    Default:
    null

    最大高度，高度默认自适应，当达到最大高度之后会隐藏内容 。

    示例：

    $("#tip").omTooltip({maxHeight: 100});

  • maxWidth

    Type:
      Number
    Default:
    null

    最大宽度，当需要根据内容自动适应宽度而又不想因为内容太多而导致太宽的情况，可以使用maxWidht设置一个最大宽度。

    示例：

    $("#tip").omTooltip({maxWidth: 200});

  • minHeight

    Type:
      Number
    Default:
    null

    最小高度。

    示例：

    $("#tip").omTooltip({minHeight: 100});

  • minWidth

    Type:
      Number
    Default:
    null

    最小宽度，当提示内容很少的时候为了样式美观显示指定的宽度。

    示例：

    $("#tip").omTooltip({minWidth: 100});

  • offset

    Type:
      Object
    Default:
    [5,5]

    提示框显示在指定位置的偏离度,提示框有两种基准位置，一是鼠标，二是目标区域, 第一个参数为top偏差值，第二个为left偏差值。

    示例：

    $("#tip").omTooltip({offset:[10,15]});

  • region

    Type:
      String
    Default:
    null

    提示框显示的区域,设置提示框显示的区域，如果不设置， 则以鼠标的当前位置为基准点显示并跟随鼠标移动， 如果设置了则以目标为基准点显示,可选值有left、right、top、bottom， 比如选择了left，则提示框永远出现在目标的左边，不会进入目标。

    示例：

    $("#tip").omTooltip({region:'bottom'});

  • showOn

    Type:
      String
    Default:
    'mouseover'

    显示提示框的触发方式,可选值有mouseover、click

    示例：

    $("#tip").omTooltip({showOn: click});

  • trackMouse

    Type:
      Boolean
    Default:
    false

    提示框是否跟随鼠标移动,默认不跟随鼠标移动，设置为true则跟随鼠标移动， 比region优先级低。

    示例：

    $("#tip").omTooltip({trackMouse: false});

  • url

    Type:
      String
    Default:
    null

    ajax加载内容的url地址,返回的内容格式dataType为html

    示例：

    $("#tip").omTooltip({url:'a.html'});

  • width

    Type:
      Number
    Default:
    'auto'

    提示框的宽度。

    示例：

    $("#tip").omTooltip({width: 100});

方法

  • hide

    Signature:
      hide()

    隐藏组件。

    示例：

    $('#mytip').omTooltip('hide');

  • show

    Signature:
      show()

    显示组件，组件只有在显示的时候才添加到dom树， 第二次显示已经存在dom树，只需要改变显示状态。
