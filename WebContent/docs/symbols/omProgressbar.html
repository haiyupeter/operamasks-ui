Class omProgressbar
  • 预览
  • 属性
  • 方法
  • 事件

进度条一般用来呈现任务完成的进度情况。
示例：

<script type="text/javascript" >
$(document).ready(function() {
    $('#selector').omProgressbar({
        value : 30
    });
});
</script>

<div id="selector" />

omProgressbar(p)

示例：

$('#selector').omProgressbar();
参数:
p - 标准config对象：{}

属性

  • text

    Type:
      String, Function
    Default:
    "{value}%"

    提示内容，默认的内容格式为{value}%，如当进度值为30时，显示的文本为30%; 并且支持方法自定义提示内容， 方法提供一个参数为当前的进度值，返回值为提示内容。

    示例：

    $("#selector").omProgressbar({text: "已完成{value}%"});

  • value

    Type:
      Number
    Default:
    0

    进度值，默认值为0，最大值为100

    示例：

    $("#selector").omProgressbar({value:50});

  • width

    Type:
      Number,String
    Default:
    auto

    设置进度条的宽度,单位为像素,默认值为"auto"自适应宽度。

    示例：

    $("#selector").omProgressbar({width: 300});

方法

  • value

    Signature:
      value(newValue)

    获取或者设置进度值。没有传入参数时，该方法为获取当前进度值，反之，则设置当前进度值。

    参数:
    newValue - Number对象 设置进度值

    示例：

    $("#selector").omProgressbar('value', '30');

事件

  • onChange

    function(newValue, oldValue, event)

    进度值改变触发事件。

    参数:
    newValue - 改变后进度值
    oldValue - 改变前进度值
    event - jQuery.Event对象

    示例

     $("#selector").omProgressbar({
     onChange: function(newValue, oldValue, event){ ... }
 });
