Class omTabs
  • 预览
  • 属性
  • 方法
  • 事件

页签布局组件，通过简单的配置展示多页签信息，同时组件提供丰富的事件支持，比如选中页签，关闭页签，添加页签等等。
支持各个页签以ajax方式加载内容；支持懒加载；支持页签滚动。
使用方式：

页面上的html标记如下 

<script type="text/javascript" >
$(document).ready(function() {
    $('#make-tab').omTabs({});
});
</script>

     <div id="make-tab">
         <ul>
             <li>
                 <a href="#tab1">Title1</a>
             </li>
             <li>
                 <a href="#tab2">Title2</a>
             </li>
             <li>
                 <a href="#tab3">Title3</a>
             </li>
         </ul>
         <div id="tab1">
             this is tab1 content
         </div>
         <div id="tab2">
             this is tab2 content
         </div>
         <div id="tab3">
             this is tab3 content
         </div>
     </div>

omTabs(p)

示例：

$('#make-tab').omTabs({width:500, height:300});
参数:
p - 标准config对象：{width:500, height:300}

属性

  • active

    Type:
      Number,String
    Default:
    0

    初始化时被激活页签的索引（从0开始计数）或者tabId。

    示例：

    $('#make-tab').omTabs({active : 1});//初始化时激活第二个页签
$('#make-tab').omTabs({active : 'tab-1'});//初始化时激活Id为'tab-1'的页签

  • autoPlay

    Type:
      Boolean
    Default:
    false

    是否自动循环切换页签。

    示例：

    $('#make-tab').omTabs({autoPlay:true});//自动切换页签

  • border

    Type:
      Boolean
    Default:
    true

    是否显示页签正文区的边框。

    示例：

    $('#make-tab').omTabs({border: false});//不显示页签正文区的边框

  • closable

    Type:
      Boolean,Array
    Default:
    false

    页签是否可关闭，当本属性为true时，所有页签都可以关闭。当属性值为数组时，只有数组中指定的index的页签可以关闭，index从0开始。

    示例：

    //页签可关闭
$('#make-tab').omTabs({closable : true});

//只有第一个和第三个页签可以关闭
$('#make-tab').omTabs({closable : [0,2]);

  • height

    Type:
      Number,String
    Default:
    'auto'

    页签布局的高度，可取值为'auto'(默认情况，不做处理)，可以取值为'fit'，表示适应父容器的大小(height:100%)，也可以直接设置height大小（单位：像素）。

    示例：

    $('#make-tab').omTabs({height: 200});

  • interval

    Type:
      Number
    Default:
    1000

    自动切换页签的时间间隔，单位为毫秒。 该属性在 switchMode 为auto时才生效。

    示例：

    $('#make-tab').omTabs({autoPlay:true, interval : 2000});//自动切换页签时，时间间隔为2s

  • lazyLoad

    Type:
      Boolean
    Default:
    false

    是否懒加载，当该属性为true时，只有在页签被单击选中时才尝试加载页签正文区。

    示例：

    $('#make-tab').omTabs({lazyLoad : true});

  • scrollable

    Type:
      Boolean
    Default:
    true

    当页签超过组件宽度时是否出现左右滚动箭头。

    示例：

    //当页签数目较多时不显示滚动箭头，将访问不到未显示的页签
$('#make-tab').omTabs({scrollable: false});

  • switchMode

    Type:
      String
    Default:
    'click'

    页签切换的模式。可为click(鼠标点击切换)，mouseover(鼠标滑过切换)。注意：当设置了autoPlay属性时，虽然组件在自动切换，此时仍可以使用鼠标点击（鼠标划过）切换页签

    示例：

    $('#make-tab').omTabs({switchMode : 'mouseover'});//鼠标划过切换页签

  • tabHeight

    Type:
      Number,String
    Default:
    25

    单个页签头部的高度，可取值为'auto'。默认为25像素。

    示例：

    $('#make-tab').omTabs({tabHeight: 'auto'});

  • tabWidth

    Type:
      Number,String
    Default:
    auto

    单个页签头部的宽度。

    示例：

    $('#make-tab').omTabs({tabWidth: 'auto'});

  • width

    Type:
      Number,String
    Default:
    'auto'

    页签布局的宽度，可取值为'auto'(默认情况，不做处理)，可以取值为'fit'，表示适应父容器的大小(width:100%)，也可以直接设置width大小（单位：像素）。

    示例：

    $('#make-tab').omTabs({width: 500});

方法

  • activate

    Signature:
      activate(n)

    选中特定的页签，触发activate事件。

    参数:
    n - 可为页签的索引（从0开始计数），或者页签的tabId

    示例：

    //激活第一个页签
$('#make-tab').omTabs('activate', 0);

  • add

    Signature:
      add(Object)

    在index处增加一个tab页签。参数为json格式的配置项。 调用该方法会触发 add事件。 配置项参数：

    1. index：新增页签的位置（从0开始计数,默认在末尾增加页签），可设置为'last'
    2. title：新增页签的标题，默认值为 'New Title' + 全局唯一字符串
    3. content：新增页签的内容，默认值为 'New Content' + 全局唯一字符串
    4. url：新增页签的数据源为url。如果同时设置了content和url，则优先使用url
    5. tabId：设置tabId，作为唯一标识，可以通过此标识唯一确定一个tab页签，tabId不能重复
    6. closable：该新增的页签是否可关闭。
    参数:
    Object - {index,title,content,url,colsable,tabId}

    示例：

    //在第一个页签的位置新增一个页签,该页签的内容是远程数据
$('#make-tab').omTabs('add', {
    index : 0,
    title : 'New Tab1',
    content : 'New Content1',
    closable : false
});

  • close

    Signature:
      close(n)

    关闭特定的页签，如果n指向当前页签，则会选中下一页签；如果当前页签是最末尾的页签，则会选中第一个页签。可以看到每关闭一个页签就会分别触发一次close事件和activate事件。

    参数:
    n - 要关闭的页签的位置（从0开始计数），或者该页签的tabId(一个全局唯一的字符串)。 如果未指定该参数，则默认关闭当前页签。

    示例：

    //关闭第一个页签
$('#make-tab').omTabs('close', 0);

  • closeAll

    Signature:
      closeAll()

    关闭所有页签，由于该操作只关注于删除所有页签，因此只会触发 onCloseAll事件，而不会逐个触发每个页签的onClose事件。

    示例：

    //关闭所有页签
$('#make-tab').omTabs('closeAll');

  • doLayout

    Signature:
      doLayout()

    对组件重新布局，主要操作是刷新页签滚动箭头。 如果有必要使用页签滚动箭头，则刷新滚动箭头的状态。如果没必要使用页签滚动箭头，则将存在的删除。

    示例：

    //对组件重新布局，如果有必要使用页签滚动箭头，则刷新滚动箭头的状态。

  • getActivated

    Signature:
      getActivated()

    返回当前选中的页签的tabId。

    返回:
    当前选中页签的tabId

    示例：

    //获取当前选中页签的tabId
var activatedTabId = $('#make-tab').omTabs('getActivated');

  • getAlter

    Signature:
      getAlter(id)

    页签索引和tabId的转换器。传入其中的一个值，获取另一个值。

    参数:
    id - 标识符
    返回:
    如果id为数字，则表示页签的索引，函数返回页签的tabId；如果id为字符串，则表示该页签的tabId，函数返回页签的索引。 如果索引不合法或者id作为tabId时找不到，则统一返回null。

    示例：

    //获取第一个页签的tabId
var tabId = $('#make-tab').omTabs('getAlter', 0);

  • getLength

    Signature:
      getLength()

    获得所有页签的数目。

    返回:
    页签的数目

    示例：

    //获取页签的总数
var total = $('#make-tab').omTabs('getLength');

  • reload

    Signature:
      reload(index, url, content)

    根据新的数据源重新加载某个页签的内容。

    参数:
    index - 页签的索引
    url - 页签为远程取数时的url，此属性优先级高于content
    content - 页签的文本内容

    示例：

    //重新加载第一个页签的内容
$('#make-tab').omTabs('reload', 0 , "./getData.html");

  • setDataSource

    Signature:
      setDataSource(index, content, url)

    (deprecated，建议用reload方法) 设置第n个页签的数据源，可为普通文本或者url。注意该方法只是会重置一个当前页签是否已被加载的标记，而不负责实际加载数据， 在非懒加载的情况下，需要手动加载数据。在懒加载的情况下，当页签被点击选中时会检查是否已经加载的标记，从而尝试重新加载内容。

    参数:
    index - 被操作页签的索引(从0开始计数)
    content - 设置了该属性则表示数据源为普通文本。
    url - 设置了该属性表示数据源是远程url，如果同时设置了content和url，则优先使用url。

    示例：

    //设置第一个页签的数据源为远程数据
 $('#make-tab').omTabs('setDataSource', {
     index : 0,
     url : './ajax/content1.html'
 });

事件

  • onActivate

    function(n, event)

    当页签被选中后执行的方法。

    参数:
    n - 选中页签的索引，从0开始计数.
    event - jQuery.Event对象

    示例

     $('#make-tab').omTabs({
     onActivate : function(n,event) {
         alert('tab ' + n + ' has been activated!');
     }
 });

  • onAdd

    function(config, event)

    当新页签被添加之后执行的方法。

    参数:
    config - 经过处理的配置项。在调用add新增页签时，传入的配置项参数可能不完整(使用默认值)，此处的config就是完整的配置项
    event - jQuery.Event对象

    示例

     $('#make-tab').omTabs({
     onAdd : function(config,event) {
         console.dir(config);
         alert('you have added a tab at position:' + config.index );
     }
 });
